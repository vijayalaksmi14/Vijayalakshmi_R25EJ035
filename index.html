// Sandbox Simulation

Try The Graphics Editor

graphics_sandbox.c

Draw in Realtime

This sandbox adapts the drawing mathematical routines implemented in the C 2D Graphics Editor. Click and drag or click endpoint coordinates to generate shapes.

1

Select a drawing tool

Choose from standard pixel plotting, lines (using Bresenham algorithms), rectangles, or circle math.

2

Interact with the Grid

Click on cells. For line, rectangle, or circle, click the starting cell and then the ending cell to render.

3

See '*' and '_' In Action

The grids print '*' for drawn outlines and '_' for background space, exactly like the C editor canvas.
